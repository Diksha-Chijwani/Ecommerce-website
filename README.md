# E-Commerce Website

## Features
- User authentication (login/register)
- Product catalog with search/filter
- Shopping cart system
- Order management
- Admin dashboard

## Tech Stack
- HTML, CSS, JavaScript
- PHP
- MySQL

## Folder Structure
- admin/ → admin panel
- controllers/ → logic handling
- models/ → database operations
- resources/ → CSS, JS, images
- sql-script/ → database setup

## How to Run
1. Move project to XAMPP htdocs
2. Import SQL file from sql-script
3. Start Apache & MySQL
4. Open localhost/public-html
